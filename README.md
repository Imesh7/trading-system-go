# Trading system using go

Create a Order process in a trading system.
 - Create a order
 - Maintain orderbook
 - Match the order with orderbook

 Tech stack used
 --------------------
 - Backend server
 - Postgres (db)
 - Kafka
 - Redis (orderbook)

## Running the Application

To run the application, use the following command:

```bash
docker-compose up
